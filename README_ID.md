# Yui WaBot

Ini adalah bot WhatsApp yang dibuat dengan Baileys.

## Instalasi

Ikuti langkah-langkah ini untuk menginstal dan menjalankan bot di platform pilihan Anda.

### Prasyarat

- [Node.js](https://nodejs.org/) (disarankan v18 atau lebih tinggi)
- [Git](https://git-scm.com/)

### Platform

#### Linux

1.  **Instal Node.js dan Git:**
    ```bash
    sudo apt update
    sudo apt install -y nodejs git
    ```

2.  **Clone repositori:**
    ```bash
    git clone https://github.com/username-anda/yui-wabot.git
    cd yui-wabot
    ```

3.  **Instal dependensi:**
    ```bash
    npm install
    ```

4.  **Konfigurasi environment:**
    Salin file `.env.example` ke `.env` dan isi nilai yang diperlukan.
    ```bash
    cp .env.example .env
    ```

5.  **Build dan jalankan bot:**
    ```bash
    npm run build
    npm start
    ```
