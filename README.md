# Yui WaBot

This is a WhatsApp bot built with Baileys.

## Installation

Follow these steps to install and run the bot on your preferred platform.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [Git](https://git-scm.com/)

### Platforms

#### Linux

1.  **Install Node.js and Git:**
    ```bash
    sudo apt update
    sudo apt install -y nodejs git
    ```

2.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/yui-wabot.git
    cd yui-wabot
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Configure the environment:**
    Copy the `.env.example` file to `.env` and fill in the required values.
    ```bash
    cp .env.example .env
    ```

5.  **Build and run the bot:**
    ```bash
    npm run build
    npm start
    ```
